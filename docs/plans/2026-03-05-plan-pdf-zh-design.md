# plan.pdf 中文化设计说明

## 目标
在尽量保持原 `plan.pdf` 版式（双栏、图表、公式位置）的前提下，生成中文正文 PDF；参考文献保留英文原文。

## 约束
- 允许小范围版式变化（段落换行、字号微调）。
- 图、表、公式尽量不变。
- 参考文献区不翻译。

## 方案（已确认）
采用方案 C：`块级替换 + 人工校正`。

1. 自动阶段：
- 解析 PDF 页面文本块坐标。
- 对正文英文块执行白底覆盖并写入中文。
- 跳过参考文献区域。

2. 人工校正阶段：
- 渲染 PNG 逐页检查。
- 修正重叠、溢出、截断。

## 产物
- 脚本：`tools/pdf_translate_overlay.py`
- 中间结果：`tmp/pdfs/plan_zh_working.pdf`、渲染图
- 最终结果：`src/plan_zh.pdf`
